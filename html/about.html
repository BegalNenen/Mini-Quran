# Tentang MiniQuran

Halaman ini menyediakan informasi mengenai [MiniQuran] yang memfasilitasi akses ke ayat-ayat Al-Qur'an melalui API publik. Tujuan kami adalah untuk menyediakan sumber yang mudah digunakan dan terpercaya bagi siapa saja yang ingin belajar dan mempelajari Al-Qur'an. Kami berharap dapat membantu dalam upaya Anda untuk meningkatkan pengetahuan dan spiritualitas Anda.

Kami menggunakan API Quran gratis dari [equran.id]. Terima kasih kepada mereka atas akses dan sumber data yang berguna.
